## 2026-04-20 — Sub-phase 1a: scaffolding + auth

`app-v2/` is now a standalone Vite + React 19 + TypeScript 6 project that builds clean (`npm run typecheck` + `npm run build` + `npm run lint` all pass). Tailwind v4 wired to `tokens.css`, which carries the full palette from design §6.1 (action tones + state tones + side-identity tints) in both light and dark modes. `index.html` has a pre-paint script that respects `prefers-color-scheme` before React mounts, preventing FOUC on dark-preference devices. Dev server runs via `npm run dev`; the manual dark-mode toggle ships in Settings (Phase 2). React-router v7 drives four base routes (`/`, `/cards`, `/community`, `/me`) plus the standalone `/s/:code`, `/u/:handle`, and 404 fallback. A `TabBar` primitive renders the bottom four-tab nav with an `aria-label="Primary"` on a `role="tablist"`, meeting §7.10's keyboard-nav requirement. `Screen` + `EmptyState` are the first two primitives; the remaining ten land as Phase 1 needs them. TanStack Query provider is wired at root with a 30s default staleTime; Zustand store (`useUiStore`) persists `dismissedBanners` + `seenHints` to `localStorage`. `useAuth()` fetches `/api/auth/me` and reconciles with the `swu.signedInHint` localStorage flag the same way v1 does. `api/auth.ts` is a thin v2 port of v1's dispatcher — imports `createSession` / `getSession` / `mergeGhostIntoRealUser` / `syncGuildMemberships` directly from the repo-root `lib/` (shared with v1, zero duplication). To make the shared-lib import typecheck without drizzle-orm's nominal-private-property types doubling across two `node_modules` trees, the root `package.json` now declares `"workspaces": ["app-v2"]` — npm hoists every shared dep, `app-v2/node_modules` stays empty. v1 still typechecks green. **Me tab** renders a minimal signed-in card (avatar + handle + sign-out) or the "Continue with Discord" CTA for signed-out. Footer pill carries commit SHA + beta badge via the same `isBetaChannel()` + Vite `define` pattern as v1. Vercel project config needed (by human, when ready to deploy): Root Directory = `app-v2`, Include files outside root = ON, Discord OAuth redirect registered for `next.swutrade.com`. Documented in `app-v2/README.md`.
